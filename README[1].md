# 英単語帳サイト

英単語学習のための Webアプリケーションです。単語の登録、テスト、復習、学習状況の可視化などの機能を提供します。

## 🌟 主な機能

### 1. 単語追加
- **共有単語**：管理者（ホスト）が追加した単語は全ユーザーに表示
- **私有単語**：ゲストでも追加可能（端末内に保存、他ユーザーには非表示）
- 連続追加機能（追加後すぐに次の番号が自動入力）
- 番号、英単語、日本語訳、例文を登録

### 2. 単語一覧
- 番号・単語・意味で検索
- 共有/私有で絞り込み
- 単語の編集・削除（権限に応じて）
- 音声読み上げ機能（英単語/例文）

### 3. テスト機能
- 番号範囲を指定してテスト
- 2つのモード：
  - **英→日**：英単語を見て日本語訳を答える
  - **日→英**：日本語訳を見て英単語を答える
- タップで答えを表示
- 自己採点（できた/できてない）
- テスト結果の記録

### 4. 復習機能
- 間違えた単語を優先的に表示
- 正答率が低い単語から復習
- 対象（共有/私有/すべて）を選択可能

### 5. 学習状況の可視化
- 総単語数、テスト回数、平均正答率
- 日別テスト回数のグラフ
- 正答率推移のグラフ

### 6. バックアップ機能
- 私有単語とテスト履歴をJSON形式でエクスポート
- バックアップファイルをインポートして復元
- 端末変更時のデータ移行に対応

## 🔐 ユーザー権限

### ゲストモード（デフォルト）
- 共有単語の閲覧
- 私有単語の追加・編集・削除
- すべての学習機能の利用

### 管理者モード
- 管理コード入力で昇格
- 共有単語の追加・編集・削除
- ゲストモードのすべての機能

**管理コード**：`admin123`（変更する場合は `js/config.js` を編集）

## 📊 データ管理

### 共有データ（サーバー保存）
- 共有単語：全ユーザーに表示
- ホスト権限状態

### ローカルデータ（端末内保存 - IndexedDB）
- 私有単語
- テスト履歴
- ⚠️ **注意**：ブラウザのデータ削除で消える可能性があります
- **推奨**：定期的にバックアップ機能でデータをエクスポート

## 🎯 使い方

### 初めて使う場合
1. サイトにアクセス
2. 「単語追加」タブで私有単語を追加
3. または「管理者になる」で共有単語を追加

### 単語を学習する
1. 「テスト」タブで範囲とモードを選択
2. 問題を見て考える
3. 「答えを表示」をタップ
4. 「できた」または「できてない」をタップ
5. テスト結果が記録される

### 苦手な単語を復習する
1. 「復習」タブを開く
2. 対象を選択して「復習開始」
3. 正答率が低い順に表示される

### データをバックアップする
1. 「バックアップ」タブを開く
2. 「データをエクスポート」でJSON形式で保存
3. 別の端末では「データをインポート」で復元

## 📡 API エンドポイント

### 共有単語
- `GET tables/shared_words`：共有単語一覧取得
- `POST tables/shared_words`：共有単語追加（ホストのみ）
- `PUT tables/shared_words/{id}`：共有単語更新（ホストのみ）
- `DELETE tables/shared_words/{id}`：共有単語削除（ホストのみ）

### ホスト状態
- `GET tables/host_status`：ホスト状態取得
- `POST tables/host_status`：ホスト状態作成
- `PUT tables/host_status/{id}`：ホスト状態更新

## 📝 番号の仕様

**方式A**を採用：
- 共有単語：共有内で通し番号（No.1〜）
- 私有単語：私有内で通し番号（No.1〜）
- テスト範囲指定時に対象（共有/私有/すべて）を選択

## ⚠️ 注意事項

1. **私有データの保持**
   - 端末のブラウザに保存されます
   - ブラウザのデータ削除、端末故障で消える可能性があります
   - 定期的にバックアップを取ることを推奨します

2. **管理コード**
   - デフォルトは `admin123`
   - 本番運用時は必ず変更してください（`js/config.js`）

3. **音声読み上げ**
   - 標準速度、en-US（アメリカ英語）で発音
   - ブラウザやOS設定により動作が異なる場合があります
